أساسيات بايثون

المتغيرات في بايثون

المتغيرات هي رموز تُستخدم لتخزين البيانات في بايثون. يمكن إعطاء المتغيرات أسماء لتمثيل القيم المختلفة.

x = 5
name = "Python"
print(x)
print(name)

أنواع البيانات البدائية في بايثون

تتضمن أنواع البيانات البدائية في بايثون:

  • int: الأعداد الصحيحة
  • float: الأعداد العشرية
  • str: النصوص
  • bool: القيم المنطقية (True أو False)
x = 10
pi = 3.14
name = "Python"
is_valid = True

العوامل في بايثون

تُستخدم العوامل لتنفيذ العمليات الحسابية، المنطقية، والمقارنات. أمثلة:

  • العوامل الحسابية: +, -, *, /
  • عوامل المقارنة: ==, !=, >, <
  • العوامل المنطقية: and, or, not

التعابير الشرطية

تُستخدم التعابير الشرطية لتنفيذ الكود بناءً على شروط معينة.

x = 10
y = 20
if x < y:
    print("x أقل من y")
else:
    print("x أكبر من أو يساوي y")

القوائم Lists

القوائم هي بنية بيانات تُستخدم لتخزين مجموعات من العناصر.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # apple

الصفوف Tuples

الصفوف تشبه القوائم، لكنها غير قابلة للتغيير.

colors = ("red", "green", "blue")
print(colors[1])  # green

القواميس Dictionaries

القواميس تخزن البيانات كأزواج مفتاح وقيمة.

person = {"name": "John", "age": 30}
print(person["name"])  # John

الحلقات التكرارية

تُستخدم الحلقات لتكرار الكود عدة مرات.

for i in range(5):
    print(i)

# While loop
x = 0
while x < 5:
    print(x)
    x += 1

بنى List Comprehension

تُستخدم List Comprehension لتبسيط إنشاء القوائم.

squares = [x**2 for x in range(5)]
print(squares)

الدوال Functions

الدوال هي كتل من الكود تُستخدم لتنفيذ مهمة معينة.

def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))

حواشي الدوال Function Annotations

تُستخدم حواشي الدوال لتوضيح أنواع المدخلات والمخرجات.

def add(x: int, y: int) -> int:
    return x + y

print(add(5, 3))

المزخرفات Decorators

المزخرفات (Decorators) في بايثون هي ميزة قوية تُستخدم لتعديل وظائف أو طرق (functions or methods) دون تغيير الكود الأساسي لتلك الوظائف. تُعتبر المزخرفات دوال تأخذ دالة كإدخال وتُرجع دالة جديدة مع وظائف إضافية.

أمثلة على المزخرفات

مثال 1: تنفيذ عمليات مختلفة على الأرقام

def inc(x):
    return x + 1

def dec(x):
    return x - 1

def calculate(operation, x):
    return operation(x)

print(calculate(inc, 5))  # Output: 6
print(calculate(dec, 4))  # Output: 3

في هذا المثال، تُمرَّر دالة كوسيط لدالة أخرى لتطبيق عملية معينة.

مثال 2: تنظيم استدعاء الدوال

def parent():
    print('This is the parent function')

def child1():
    print('This is the child1 function')

def child2():
    print('This is the child2 function')

child1()
child2()
parent()

هذا المثال يُظهر تنظيم استدعاءات الدوال وترتيب تنفيذها.

مثال 3: المزخرفات مع الشروط

def vot(age):
    def allow():
        print('Person can vote')

    def not_allow():
        print('Person cannot vote')

    if age > 18:
        return allow()
    else:
        return not_allow()

# Example usage:
# vot(19)  # Output: Person can vote
# vot(11)  # Output: Person cannot vote

في هذا المثال، بناء الدالة يعتمد على شرط معين لتحديد أي دالة يتم استدعاؤها.

مثال 4: استخدام المزخرفات لإضافة تأثير بصري

def decorator(function):
    def inner():
        print("=" * 20)
        function()
        print("=" * 20)
    return inner

@decorator
def write():
    print("Hello, World!")

write()

هذا المثال يُظهر كيف يمكن استخدام المزخرفات لإضافة تأثير بصري عند استدعاء الدالة.
